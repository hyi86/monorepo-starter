# Virtual List 예제

- 대용량 리스트/그리드의 렌더링 성능 최적화 실전 예제
- **Fixed Row/Column/Grid**: 고정 크기 행/열/그리드의 가상 스크롤 구현
- **Fixed Masonry (Vertical/Horizontal)**: Masonry 레이아웃의 고정형 가상화
- **Dynamic Row/Column/Grid**: 동적(가변) 크기 행/열/그리드의 가상화 처리
- **Infinite Scroll**: 무한 스크롤 기반의 데이터 로딩 및 가상화
- **Sortable Row/Column**: 행/열의 드래그&드롭 정렬과 가상화의 결합
- **Virtual List & Parallel Routes**: 페이지 이동 시, 상태 유지 및 사용자 경험 최적화
- 각 예제는 실제 스크롤/렌더링 최적화, 드래그&드롭, Masonry, 무한 스크롤 등  
  실무에서 자주 쓰이는 고급 패턴을 포함
- `@tanstack/virtual` 등 최신 라이브러리와 커스텀 훅 활용
