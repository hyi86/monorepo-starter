# Table 예제

- 다양한 Table UI/UX 패턴 실전 예제
- `Tanstack Table`, `Shadcn/ui`, `Nuqs` 등 다양한 라이브러리 활용
- **Server Basic**: 서버 컴포넌트 기반의 기본 테이블 렌더링
- **Server Controls**: 서버 컴포넌트 기반의 필터, 정렬, 페이지네이션 등 다양한 컨트롤 기능
- **Client Full**: 클라이언트 컴포넌트 기반의 고급 테이블
  - 필터, 정렬, 페이지네이션, 행/열 선택, 열 크기 조정, 열/행 고정, 가상화, 드래그&드롭 등 데이터 그리드
